#  Сервис управления рассылками

Веб-приложение на Django для создания и управления email-рассылками с авторизацией, правами доступа и кешированием.

---

##  Описание проекта

Этот сервис позволяет:
- Создавать, просматривать, редактировать и удалять рассылки
- Управлять получателями и шаблонами сообщений
- Отправлять письма по расписанию или вручную
- Отслеживать статус отправки (успешно/не успешно)
- Разграничивать доступ для обычных пользователей и менеджеров

---

##  Технологии

- **Python** 3.14+
- **Django** 6.0+
- **SQLite** (по умолчанию, можно заменить на PostgreSQL)
- **Bootstrap** (для интерфейса)
- **Git** и **GitHub**

##  Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/segaskypro/mailing-service.git
cd mailing-service

2. Создайте виртуальное окружение
bash
python -m venv venv
source venv/bin/activate      # для Linux/Mac
# или
venv\Scripts\activate         # для Windows
3. Установите зависимости
bash
pip install -r requirements.txt
4. Примените миграции
bash
python manage.py migrate
5. Создайте суперпользователя (администратора)
bash
python manage.py createsuperuser
6. Запустите сервер
bash
python manage.py runserver
7. Откройте в браузере
Главная страница: http://127.0.0.1:8000

Админка: http://127.0.0.1:8000/admin

```

## Функциональность
Основные модели
Recipient — получатель рассылки (email, ФИО, комментарий)

Message — шаблон письма (тема, тело)

Mailing — рассылка (даты, статус, сообщение, получатели, владелец)

Attempt — попытка отправки (дата, статус, ответ сервера)

Отправка писем
bash
python manage.py shell
python
from mailing.services import send_mailing
send_mailing(1)  # где 1 — ID рассылки
Запуск тестов
bash
python manage.py test mailing
